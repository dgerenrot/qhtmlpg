
	USAGE:

	./qhtmlpg [OPTIONS] <TagPatternString>

	Quickly specify the structure of your HTML page on the command line!
	The <TagPatternString> determines the nesting of the tags with square
	brackets.
	
	The correct doctype, html head and title tags are also included.

	For example, if <TagPatternString> is '[]', then the output BODY contains
	a single pair of open/close DIV tags. Some other tags are also possible.

	For full usage info, specify the -? option.
